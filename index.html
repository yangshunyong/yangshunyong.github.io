Introduction

This is a tool to analyze/set each bit value of an hexmial data.

::You can select data width (16, 32, 64 bits, etc) in Data Width.

::You can copy data from analyze to set by clicking "Copy from Aanalyzed"

::You can display consecutive bits value by inputing range in "Select Some Bits"

Data Width

Input Heximal Value to Analyzer (e.g. ABCD1234)

Select Some Bits: e.g. select bit 5 to 10 by typing 5, 10

Selected Value: 0x0

Click to Set Each Bit (You can copy data from analyzed above also)
